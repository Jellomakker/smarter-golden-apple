# Smarter Eat (Fabric)

`Smarter Eat` is a Fabric mod for Minecraft `1.21.5+` (tested on 1.21.8).

## Behavior

- Right-click once with food to queue and eat 1 item.
- Right-click twice to queue and eat 2 items, and so on.
- Left-click cancels the queue.
- Switching to a different food/item cancels the queue.

## Build

Run:

```bash
./gradlew build
```

Built jars are in `build/libs`.
